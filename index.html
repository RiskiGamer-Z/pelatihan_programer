Pelatihan Programmer Komputer

Bootcamp intensif 15 pertemuan berbasis Project-Based Learning.

Tujuan Pelatihan

Selamat datang di platform pembelajaran Programmer Komputer BLK! Dalam program intensif ini, Anda akan belajar dari nol hingga mahir membuat sistem informasi yang fungsional dan aman.

🎯 Output Tersertifikasi: Anda diwajibkan menyelesaikan 3 proyek akhir: Website Profil Pribadi, Aplikasi Pendataan Sederhana, dan Sistem Informasi Pendaftaran Sekolah CRUD lengkap dengan basis data MySQL.

Skema 15 Pertemuan

  • Pertemuan 1-12: Materi inti, project bertahap, dan simulasi CRUD.
  • Pertemuan 13-15: Ujian output 1 sampai 3.
  • Pendataan siswa mulai dibangun sejak pertemuan 9 agar waktu praktik lebih panjang.

Persiapan Tools

Untuk mengikuti materi ini, pastikan komputer Anda telah ter-install perangkat lunak berikut:

  • Web Browser: Microsoft Edge, Google Chrome, atau Mozilla Firefox.
  • Text Editor: Visual Studio Code atau Sublime Text (disarankan).
  • Web Server Lokal: XAMPP (untuk menjalankan Apache dan MySQL).
  • Package Manager: Composer (akan digunakan di Fase 5).

Cara Menggunakan Modul Ini

Klik menu di sidebar sebelah kiri untuk mulai membaca materi. Setiap pertemuan didesain untuk satu hari pembelajaran (5 Jam Pelajaran / 300 menit). Jangan lupa untuk selalu mengetik ulang kode yang ada di modul (hindari *copy-paste*) agar otot memori jari Anda terbiasa dengan sintaks pemrograman!
