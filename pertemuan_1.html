Pertemuan 1: Konsep Dasar Web & Pengenalan HTML

Memahami internet, protokol web, dan struktur rangka sebuah website.

A. Dasar-Dasar Website

Internet merupakan "kependekan dari kata internetwork", yang berarti rangkaian komputer yang terhubung menjadi beberapa rangkaian jaringan global menggunakan TCP/IP sebagai protokol. Secara umum internet adalah pertukaran informasi bebas tanpa batasan.

Istilah Penting di Internet

  • WWW (World Wide Web): Kumpulan web server diseluruh dunia yang melayani data dan informasi.
  • Website/Situs Web: Alamat di WWW yang memiliki kumpulan halaman web.
  • Browser: Aplikasi untuk berselancar di dunia internet (Cth: Chrome, Edge).
  • URL (Uniform Resource Locator): Alamat spesifik sebuah halaman. (Cth: http://google.com).
  • HTTP/HTTPS: Protokol identifikasi lokasi web tempat data ditransfer.
📝 Web Server: Perangkat lunak yang merespons permintaan browser (HTTP). Contoh terkenalnya adalah Apache (yang tergabung dalam aplikasi XAMPP).

B. Pengertian HTML

HTML (Hypertext Markup Language) adalah bahasa *markup* standar untuk menampilkan konten di web. HTML bukanlah "bahasa pemrograman logika", melainkan bahasa markup struktural untuk memberitahu browser bagaimana menampilkan teks atau gambar.

Sebuah file HTML selalu berisi elemen tag yang diawali lambang < dan diakhiri >.

Struktur Dasar HTML

Struktur kerangka dasar wajib dari halaman website adalah seperti ini:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Web Anda</title>
  </head>
  <body>
    Konten atau isi website ditampilkan di area BODY ini.
  </body>
</html>
  • <html>: Tag pembuka halaman web keseluruhan.
  • <head>: Bagian "kepala", tempat Anda menaruh judul dan konfigurasi (tidak tampil langsung di layar putih browser).
  • <title>: Judul halaman yang akan muncul di tab browser.
  • <body>: Bagian "badan", tempat semua foto, teks, dan tombol utama Anda diletakkan!

Tag-Tag HTML Mandiri yang Sering Dipakai

Kebanyakan tag perlu memiliki pasangan penutup, contoh: <h1> ditutup dengan </h1>. Namun ada juga yang berdiri sendiri:

  • <br>: Baris baru (Enter / Line Break).
  • <hr>: Garis datar pembatas.

💻 Praktik & Tugas!

Sekarang uji pemahaman Anda. Praktikkan langsung instruksi di bawah ini dengan menyalakan Text Editor (Sublime / VS Code):

  1. Buat file baru di editor Anda.
  2. Ketik kode HTML struktur dasar di atas, lalu tambahkan "Selamat Datang di Website Saya" di dalam bagian <body>.
  3. Gunakan tag <br> untuk memisahkan baris paragraf kedua Anda.
  4. Simpan file (Save As) dengan nama contoh1.html di desktop Anda. Pastikan pada "Save as type" dipilih "All Files".
  5. Buka file tersebut dengan mengeklik ganda file contoh1.html. Munculkah tulisan tersebut di browser Edge/Chrome Anda?

Jika berhasil, selamat! Anda sudah menulis baris kode Website pertama Anda! 🎉 Lanjut ke Pertemuan 2 untuk elemen visual lainnya.
