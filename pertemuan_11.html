Fase 3: Database & Integrasi

Pertemuan 11: Membuat Database, Tabel, Koneksi, dan Struktur CRUD

Target pertemuan ini: siswa bisa membuat database di phpMyAdmin, membuat tabel dengan 2 cara, membuat koneksi sederhana, menyusun struktur file rapi, lalu menjalankan alur CRUD dasar.

A. Tujuan Pembelajaran

  • Siswa bisa membuat database dan tabel melalui phpMyAdmin.
  • Siswa memahami 2 cara membuat tabel: manual (form) dan SQL.
  • Siswa memahami fungsi kolom umum tabel seperti id, nama, dan lainnya.
  • Siswa bisa membuat file config.php untuk koneksi sederhana ke database.
  • Siswa bisa menata struktur file project CRUD dengan rapi.
  • Siswa memahami alur kerja CRUD dari tambah data sampai hapus data.

B. Alur Besar Pertemuan 11

  1. Buat database pendaftaran_siswa di phpMyAdmin.
  2. Buat tabel calon_siswa dengan 2 metode (manual dan SQL).
  3. Pahami arti setiap kolom tabel.
  4. Buat file koneksi config.php.
  5. Susun struktur file CRUD.
  6. Jalankan contoh halaman: tampil data, tambah data, edit, hapus.

C. Langkah 1 - Membuat Database di phpMyAdmin

  1. Aktifkan Apache dan MySQL di Laragon/XAMPP.
  2. Buka phpMyAdmin di browser.
  3. Klik tombol New (kiri atas).
  4. Isi nama database: pendaftaran_siswa.
  5. Klik Create.

Hasil yang diharapkan: database pendaftaran_siswa muncul di panel kiri.

D. Langkah 2 - Membuat Tabel dengan 2 Cara

Cara 1: Manual lewat Form phpMyAdmin

  1. Klik database pendaftaran_siswa.
  2. Pada bagian Create table, isi nama tabel: calon_siswa.
  3. Isi jumlah kolom: 6, lalu klik Go.
  4. Isi kolom satu per satu sesuai tabel desain di bawah.
  5. Set id sebagai PRIMARY dan centang A_I (Auto Increment).
  6. Klik Save.

Cara 2: Lewat SQL

Buka tab SQL, lalu jalankan query berikut:

CREATE TABLE calon_siswa (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100) NOT NULL,
    kelas VARCHAR(30) NOT NULL,
    jurusan VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Hasil yang diharapkan: tabel calon_siswa berhasil dibuat.

E. Mengenal Fungsi Tiap Kolom Tabel

Kolom Tipe Data Wajib? Fungsi
id INT AUTO_INCREMENT PRIMARY KEY Ya Nomor unik tiap data. Dipakai untuk edit/hapus data tertentu.
nama VARCHAR(100) Ya (NOT NULL) Menyimpan nama siswa.
kelas VARCHAR(30) Ya (NOT NULL) Menyimpan kelas siswa (contoh: XI RPL 1).
jurusan VARCHAR(50) Ya (NOT NULL) Menyimpan jurusan siswa (contoh: RPL).
email VARCHAR(100) Ya (NOT NULL) Menyimpan email siswa.
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP Otomatis Mencatat waktu saat data dibuat.

Catatan: kolom bisa disesuaikan kebutuhan sekolah, tetapi id sangat disarankan selalu ada sebagai kunci utama.

F. Langkah 3 - Membuat Koneksi Sederhana (config.php)

Mulai dari versi paling dasar dulu agar siswa paham alurnya:

<?php
$host = 'localhost';
$db = 'pendaftaran_siswa';
$user = 'root';
$pass = '';

$koneksi = mysqli_connect($host, $user, $pass, $db);

if ($koneksi) {
    echo 'Koneksi database berhasil.';
} else {
    echo 'Koneksi gagal: ' . mysqli_connect_error();
}
?>

Penjelasan singkat:

  • Jika koneksi berhasil, muncul teks Koneksi database berhasil.
  • Jika koneksi gagal, muncul teks error agar siswa tahu sumber masalah.
  • Error umum: MySQL belum aktif, nama database salah, user/password salah.

G. Struktur File CRUD yang Rapi

pendaftaran_siswa_app/
├── config.php
├── index.php
├── tambah.php
├── edit.php
├── update.php
├── hapus.php
├── style.css
└── assets/
    └── script.js

Fungsi tiap file:

  • config.php: menyimpan koneksi database (dipakai ulang).
  • index.php: menampilkan data siswa (READ).
  • tambah.php: form + proses tambah data (CREATE).
  • edit.php dan update.php: ubah data (UPDATE).
  • hapus.php: hapus data siswa (DELETE).
  • style.css: pengaturan tampilan halaman.
  • assets/script.js: validasi kecil di sisi browser.

H. Contoh Kode Inti CRUD (Sederhana)

1. config.php

<?php
$host = 'localhost';
$db = 'pendaftaran_siswa';
$user = 'root';
$pass = '';

$koneksi = mysqli_connect($host, $user, $pass, $db);

if (!$koneksi) {
    die('Koneksi gagal: ' . mysqli_connect_error());
}
?>

2. index.php (READ - menampilkan data)

<?php
require 'config.php';
$query = mysqli_query($koneksi, "SELECT * FROM calon_siswa ORDER BY id DESC");
?>
<!DOCTYPE html>
<html lang="id">

    <meta charset="UTF-8">
    <title>Data Calon Siswa</title>

</head>

            <th>Nama</th>
            <th>Kelas</th>
            <th>Jurusan</th>
            <th>Email</th>

        <?php $no = 1; while ($row = mysqli_fetch_assoc($query)): ?>
            <tr>

3. tambah.php (CREATE - form dan simpan)

<?php
require 'config.php';

if (isset($_POST['simpan'])) {
    $nama = $_POST['nama'];
    $kelas = $_POST['kelas'];
    $jurusan = $_POST['jurusan'];
    $email = $_POST['email'];

    mysqli_query($koneksi, "INSERT INTO calon_siswa (nama, kelas, jurusan, email) VALUES ('$nama', '$kelas', '$jurusan', '$email')");
    header('Location: index.php');
    exit;
}
?>
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Tambah Siswa</title>
</head>
<body>
    <h2>Tambah Data Siswa</h2>
    <form method="POST">
        <input type="text" name="nama" placeholder="Nama" required><br><br>
        <input type="text" name="kelas" placeholder="Kelas" required><br><br>
        <input type="text" name="jurusan" placeholder="Jurusan" required><br><br>
        <input type="email" name="email" placeholder="Email" required><br><br>
        <button type="submit" name="simpan">Simpan</button>
    </form>
</body>
</html>

4. edit.php (UPDATE - halaman form edit)

<?php
// edit.php
require 'config.php';

if (!isset($_GET['id'])) {
    die('ID tidak ditemukan di URL.');
}

$id = (int) $_GET['id'];
$data = mysqli_query($koneksi, "SELECT * FROM calon_siswa WHERE id=$id");

if (!$data || mysqli_num_rows($data) === 0) {
    die('Data siswa tidak ditemukan.');
}

$row = mysqli_fetch_assoc($data);
?>
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Edit Siswa</title>
</head>
<body>
    <h2>Edit Data Siswa</h2>
    <form method="POST" action="update.php">
        <input type="hidden" name="id" value="<?php echo $row['id']; ?>">
        <input type="text" name="nama" value="<?php echo htmlspecialchars($row['nama']); ?>" required><br><br>
        <input type="text" name="kelas" value="<?php echo htmlspecialchars($row['kelas']); ?>" required><br><br>
        <input type="text" name="jurusan" value="<?php echo htmlspecialchars($row['jurusan']); ?>" required><br><br>
        <input type="email" name="email" value="<?php echo htmlspecialchars($row['email']); ?>" required><br><br>
        <button type="submit" name="update">Update</button>
        <a href="index.php">Batal</a>
    </form>
</body>
</html>

5. update.php (UPDATE - proses simpan perubahan)

<?php
require 'config.php';

if ($_SERVER['REQUEST_METHOD'] !== 'POST') {
    die('Akses tidak valid.');
}

$id = isset($_POST['id']) ? (int) $_POST['id'] : 0;
$nama = $_POST['nama'] ?? '';
$kelas = $_POST['kelas'] ?? '';
$jurusan = $_POST['jurusan'] ?? '';
$email = $_POST['email'] ?? '';

if ($id <= 0 || $nama === '' || $kelas === '' || $jurusan === '' || $email === '') {
    die('Data update tidak lengkap.');
}

$sql = "UPDATE calon_siswa SET nama='$nama', kelas='$kelas', jurusan='$jurusan', email='$email' WHERE id=$id";
$update = mysqli_query($koneksi, $sql);

if ($update) {
    header('Location: index.php?pesan=update-berhasil');
    exit;
} else {
    echo 'Update gagal: ' . mysqli_error($koneksi);
}
?>

Kenapa kadang tombol Edit terasa tidak berfungsi? Biasanya karena id tidak terkirim, data berdasarkan id tidak ditemukan, atau query update gagal tetapi tidak ditampilkan pesannya. Versi ini menampilkan titik gagalnya agar mudah diperbaiki.

6. hapus.php (DELETE)

<?php
require 'config.php';
$id = (int) $_GET['id'];

mysqli_query($koneksi, "DELETE FROM calon_siswa WHERE id=$id");
header('Location: index.php');
exit;
?>

Catatan pengajar: contoh di atas dibuat sesederhana mungkin untuk memahami alur. Pada pertemuan berikutnya, query akan ditingkatkan agar lebih aman dengan prepared statement.

I. Latihan Praktik (4 Soal)

  1. Latihan 1: Buat database pendaftaran_siswa dan tabel calon_siswa lewat cara manual phpMyAdmin.

  4. Latihan 4: Susun struktur file CRUD lalu jalankan halaman index.php untuk menampilkan data.
