A. Pengertian JavaScript dan Fungsinya

JavaScript (JS) adalah bahasa pemrograman yang membuat halaman web menjadi interaktif. Jika HTML adalah kerangka dan CSS adalah desain, maka JavaScript adalah logika/perilaku.

Contoh fungsi JavaScript di website:

  • Menangani klik tombol, input form, dan menu navigasi.
  • Mengubah teks, warna, dan tampilan elemen tanpa reload halaman.
  • Memvalidasi data sebelum dikirim.
  • Membuat fitur portofolio interaktif (dark mode, filter proyek, dll).
Prasyarat: Siswa sudah memahami HTML dan CSS dari pertemuan 1-3.

B. Dasar JavaScript yang Wajib Dipahami

1) Cara Menulis JavaScript

  • Inline: langsung di atribut elemen (contoh: onclick).
  • Internal: di dalam tag <script> pada file HTML.
  • External: file terpisah .js lalu dipanggil dengan <script src="...">.

2) Tipe Data Dasar

  • string: teks, contoh "Andi".
  • number: angka, contoh 16, 3.14.
  • boolean: logika benar/salah, nilai true atau false.
  • array: kumpulan data, contoh ["HTML", "CSS", "JS"].
  • object: data berpasangan key-value, contoh {nama: "Andi", umur: 16}.

3) Variabel

  • const: nilai tetap, tidak diubah lagi.
  • let: nilai bisa diubah.

4) DOM dan Event

  • DOM adalah representasi elemen HTML yang bisa diakses JS.
  • Event adalah aksi user seperti klik, ketik, submit.
  • addEventListener adalah cara standar untuk menangani event.

C. Contoh Kode + Penjelasan Tiap Kode

Contoh 1: JavaScript Aktif dan Variabel Dasar

<script>
  const nama = "Rani";
  let umur = 16;
  let aktifBelajar = true;

  console.log("Nama:", nama);
  console.log("Umur:", umur);
  console.log("Aktif belajar:", aktifBelajar);
</script>

Penjelasan kode:

  • const nama = "Rani"; membuat variabel tetap bertipe string.
  • let umur = 16; membuat variabel angka yang bisa diubah.
  • let aktifBelajar = true; membuat variabel boolean.
  • console.log(...) menampilkan data di Console browser untuk debugging.

Contoh 2: Mengubah Isi Teks dengan DOM

<h2 id="judul">Judul Lama</h2>
<button id="ubahJudul">Ubah Judul</button>

<script>
  const judul = document.getElementById("judul");
  const tombol = document.getElementById("ubahJudul");

  tombol.addEventListener("click", function () {
    judul.textContent = "Website Portofolio Siswa XI";
  });
</script>

Penjelasan kode:

  • document.getElementById("judul") memilih elemen berdasarkan id judul.
  • addEventListener("click", ...) menjalankan fungsi saat tombol diklik.
  • textContent mengganti isi teks elemen tanpa menulis HTML baru.

Contoh 3: Validasi Input Sederhana

<input id="nama" type="text" placeholder="Masukkan nama" />
<button id="cek">Cek</button>
<p id="hasil"></p>

<script>
  const nama = document.getElementById("nama");
  const cek = document.getElementById("cek");
  const hasil = document.getElementById("hasil");

  cek.addEventListener("click", function () {
    if (nama.value === "") {
      hasil.textContent = "Nama tidak boleh kosong.";
      hasil.style.color = "crimson";
      return;
    }

    hasil.textContent = "Halo, " + nama.value + "!";
    hasil.style.color = "green";
  });
</script>

Penjelasan kode:

  • nama.value membaca isi dari elemen input.
  • if (nama.value === "") mengecek apakah input kosong.
  • return; menghentikan proses jika kondisi salah.
  • hasil.style.color memberi warna teks secara dinamis.

Contoh 4: Toggle Dark Mode

<style>
  .gelap {
    background-color: #111827;
    color: #f9fafb;
  }
</style>

<button id="modeBtn">Ganti Mode</button>

<script>
  const modeBtn = document.getElementById("modeBtn");
  modeBtn.addEventListener("click", function () {
    document.body.classList.toggle("gelap");
  });
</script>

Penjelasan kode:

  • .gelap adalah class CSS untuk tema gelap.
  • document.body.classList.toggle("gelap") menambah atau menghapus class gelap setiap kali tombol diklik.
  • Dengan cara ini, logika JS dipisah dari desain CSS dan lebih rapi.

G. Materi Tambahan: Publish Portofolio ke GitHub (Akun Masing-Masing)

Setelah portofolio selesai, langkah berikutnya adalah mem-publish website agar bisa diakses online. Setiap siswa wajib publish ke akun GitHub masing-masing.

1) Persiapan

  • Pastikan sudah punya akun GitHub pribadi.
  • Pastikan file utama portofolio bernama index.html.
  • Simpan semua file terkait (HTML, CSS, JS, gambar) dalam satu folder proyek.

2) Buat Repository Baru di GitHub

  1. Login ke GitHub, klik tombol New repository.
  2. Isi nama repository, contoh: portofolio-nama-kamu.
  3. Pilih Public.
  4. Klik Create repository.

3) Upload Proyek dari Komputer (Via Git)

Buka terminal di folder portofolio, lalu jalankan perintah berikut:

git init
git add .
git commit -m "Upload portofolio pertama"
git branch -M main
git remote add origin https://github.com/USERNAME/portofolio-nama-kamu.git
git push -u origin main

Catatan: ganti USERNAME dengan username GitHub masing-masing.

4) Aktifkan GitHub Pages

  1. Buka repository di GitHub.
  2. Masuk ke menu Settings → Pages.
  3. Pada bagian Build and deployment, pilih:
  • Source: Deploy from a branch
  • Branch: main
  • Folder: / (root)
  1. Klik Save.
  2. Tunggu 1-3 menit sampai link website muncul.

5) Format Link Website Siswa

Link website biasanya berbentuk:

https://username.github.io/portofolio-nama-kamu/

6) Checklist Penilaian Publish

  • Link website bisa dibuka dari perangkat lain.
  • Tidak ada gambar/CSS/JS yang rusak (broken).
  • Fitur JavaScript tetap berjalan (dark mode, show/hide skill, validasi form).
  • Repository berisi commit minimal 2 kali (upload awal + revisi).

7) Troubleshooting Cepat

  • Jika halaman kosong, cek apakah file utama bernama index.html.
  • Jika style hilang, cek path file CSS/JS apakah sesuai folder.
  • Jika git push ditolak, jalankan git pull origin main --rebase lalu push lagi.
  • Jika link belum aktif, tunggu beberapa menit lalu refresh halaman Pages.
