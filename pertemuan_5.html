Pertemuan 5: Menggunakan XAMPP, VS Code, dan PHP Dasar

Mulai sekarang siswa belajar membuat web yang berjalan melalui server lokal, bukan hanya tampil di browser. Fokus pertemuan ini adalah menyiapkan alat kerja dan memahami PHP pertama kali.

A. Apa Itu PHP dan Mengapa Butuh XAMPP?

PHP adalah bahasa pemrograman server-side yang dipakai untuk memproses data di server sebelum hasilnya dikirim ke browser. PHP sangat cocok untuk membuat login, data form, dashboard, dan website dinamis.

Berbeda dengan HTML, CSS, dan JavaScript yang bisa langsung dibuka di browser, file PHP harus dijalankan melalui web server. Karena itu kita memakai XAMPP sebagai server lokal di laptop.

  • Apache menjalankan website lokal.
  • PHP memproses kode server-side.
  • MySQL dipakai untuk menyimpan data pada pertemuan berikutnya.
  • VS Code dipakai untuk menulis dan mengedit kode.
Inti materi: VS Code adalah tempat menulis kode, XAMPP adalah tempat menjalankan kode PHP, dan browser adalah tempat melihat hasilnya.

B. Persiapan Tools Kerja

1) Buka VS Code

  • Buka folder proyek dari VS Code dengan menu File → Open Folder.
  • Gunakan folder di dalam C:\xampp\htdocs\.
  • Contoh nama folder proyek: portofolio-pertama.

2) Jalankan XAMPP

  • Buka XAMPP Control Panel.
  • Klik Start pada Apache.
  • Jika belum ada database, MySQL belum perlu dijalankan pada pertemuan ini.

3) Aturan Folder

Semua project PHP harus disimpan di folder C:\xampp\htdocs\ supaya bisa diakses melalui alamat localhost.

C:\xampp\htdocs\portofolio-pertama\index.php

Untuk membukanya di browser, gunakan alamat:

http://localhost/portofolio-pertama/

C. Struktur Dasar File PHP

File PHP biasanya diawali dengan tag pembuka <?php dan diakhiri tag penutup ?>. Di dalamnya kita bisa menulis perintah seperti echo, variabel, kondisi, dan perulangan.

<?php
echo "Halo dari PHP";
?>

Penjelasan:

  • <?php menandai bahwa isi berikutnya adalah kode PHP.
  • echo digunakan untuk menampilkan teks ke browser.
  • ?> menutup blok PHP.

D. Contoh Kode PHP Pertama

Contoh 1: Menampilkan teks

<?php
echo "Selamat datang di pertemuan 5";
?>
  • echo mencetak teks langsung ke halaman.
  • Tanda petik dipakai karena isi yang ditampilkan adalah string.

Contoh 2: Variabel dan gabung teks

<?php
$nama = "Rani";
$kelas = "XI RPL";

echo "Nama saya " . $nama . " dan saya kelas " . $kelas;
?>
  • Tanda $ dipakai untuk membuat variabel.
  • string adalah data teks.
  • Titik . dipakai untuk menggabungkan teks di PHP.

Contoh 3: Perhitungan sederhana

<?php
$a = 5;
$b = 10;
$hasil = $a + $b;

echo "Hasil penjumlahan = " . $hasil;
?>
  • $a dan $b menyimpan angka.
  • + digunakan untuk penjumlahan.
  • Hasil disimpan ke variabel $hasil.

E. Soal Latihan Koding + Kunci Jawaban

Kerjakan latihan berurutan. Buka setiap file di browser melalui http://localhost/kunci_jawaban/pertemuan_5/latihan_*.php. Atau download file lalu jalankan di komputer masing-masing.

Soal 1 - Echo Dasar dan Variabel String

Tampilkan data pribadi (nama, sekolah, kelas) menggunakan variabel dan echo dengan penggabungan teks.

Soal 2 - Gabung String dengan Benar

Buat biodata lengkap dengan menggabungkan nama depan, nama belakang, umur, dan hobi menggunakan titik (.).

Soal 3 - Operasi Aritmatika Dasar

Lakukan perhitungan penjumlahan, pengurangan, perkalian, dan pembagian dari dua angka.

Soal 4 - Perhitungan Kompleks dengan Output Rapi

Buat program hitung rata-rata nilai 3 mata pelajaran dan tentukan status lulus/remidi.

Soal 5 - Aplikasi Kalkulator Sederhana

Buat struk pembelian dengan perhitungan subtotal, diskon, dan total bayar menggunakan tabel HTML.

F. Cara Menjalankan PHP di VS Code dan XAMPP

  1. Buka XAMPP dan hidupkan Apache.
  2. Buka folder proyek di VS Code.
  3. Buat file index.php.
  4. Tulis kode PHP di dalam file tersebut.
  5. Simpan file lalu buka browser.
  6. Ketik alamat project di localhost, contoh http://localhost/portofolio-pertama/.
  7. Jika output belum muncul, cek kembali nama file, lokasi folder, dan status Apache.
Kesalahan umum: file PHP dibuka dengan double click dari folder biasa. Cara yang benar adalah lewat localhost, bukan langsung dari file explorer.

G. Latihan Praktik di Kelas

  1. Buat folder project baru di C:\xampp\htdocs\ dengan nama latihan_php1.
  2. Buat file index.php di dalam folder tersebut.
  3. Tampilkan nama, kelas, dan umur menggunakan echo.
  4. Buat dua variabel angka lalu tampilkan hasil penjumlahannya.
  5. Buka hasilnya melalui browser menggunakan alamat http://localhost/latihan_php1/.
