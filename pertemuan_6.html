Pertemuan 6: Percabangan If Else di PHP

Di pertemuan ini siswa belajar membuat keputusan pada program PHP menggunakan if, else, dan elseif.

A. Konsep Dasar Percabangan

Percabangan adalah cara program memilih aksi berdasarkan syarat tertentu. Jika syarat bernilai true, program menjalankan blok A. Jika false, program menjalankan blok B.

Fungsi percabangan di project nyata:

  • Menentukan lulus atau tidak lulus berdasarkan nilai.
  • Menentukan diskon berdasarkan total belanja.
  • Menentukan status akun (aktif/nonaktif).
  • Menentukan hak akses (admin atau user).

B. Operator Perbandingan & Logika

Sebelum menulis if, kita harus tahu cara membandingkan nilai. Hasilnya selalu boolean: true atau false.

  • == sama dengan.
  • != tidak sama dengan.
  • > lebih besar dari.
  • < lebih kecil dari.
  • >= lebih besar atau sama dengan.
  • <= lebih kecil atau sama dengan.

Operator logika:

  • && (dan): semua syarat harus benar.
  • || (atau): cukup salah satu syarat benar.
  • ! (negasi): membalik true menjadi false, atau sebaliknya.

C. Struktur If Else dan Elseif

Gunakan struktur berikut untuk membuat keputusan:

<?php
if ($kondisi) {
    // aksi jika benar
} else {
    // aksi jika salah
}
?>

Jika kondisi lebih dari dua pilihan, gunakan elseif:

<?php
$nilai = 78;

if ($nilai >= 90) {
    echo "Grade A";
} elseif ($nilai >= 75) {
    echo "Grade B";
} else {
    echo "Grade C";
}
?>
Catatan penting: Perhatikan kurung (), kurung kurawal {}, dan titik koma ; agar tidak terjadi syntax error.

D. Contoh Kode + Penjelasan

Contoh 1: If Else Sederhana

<?php
$umur = 17;

if ($umur >= 17) {
    echo "Anda boleh membuat SIM.";
} else {
    echo "Anda belum cukup umur.";
}
?>
  • Jika $umur minimal 17, output izin membuat SIM.
  • Jika tidak, output belum cukup umur.

Contoh 2: If Elseif Else (3 kondisi)

<?php
$nilai = 82;

if ($nilai >= 90) {
    echo "Grade A";
} elseif ($nilai >= 75) {
    echo "Grade B";
} else {
    echo "Grade C";
}
?>
  • if untuk kondisi pertama.
  • elseif untuk kondisi lanjutan.
  • else untuk kondisi sisa jika semua gagal.

Contoh 3: Operator Logika

<?php
$nilai = 80;
$kehadiran = 90;

if ($nilai >= 75 && $kehadiran >= 80) {
    echo "Lulus";
} else {
    echo "Belum lulus";
}
?>
  • && berarti kedua syarat wajib benar.
  • Jika salah satu syarat gagal, hasilnya masuk ke else.

  4. Soal 4: Gunakan operator logika untuk cek kelulusan dari nilai dan kehadiran.

  6. Soal 6: Buat program cek bilangan: jika angka lebih dari 0 tampilkan "Positif", jika kurang dari 0 tampilkan "Negatif", jika sama dengan 0 tampilkan "Nol".

F. Praktik Menjalankan File Latihan

  1. Jalankan Apache dari XAMPP.
  2. Pastikan file latihan ada di folder project.
  3. Buka browser dengan alamat localhost, contoh: http://localhost/learning_platform/kunci_jawaban/pertemuan_6/latihan_1.php.
  4. Ubah nilai variabel untuk menguji kondisi true dan false.
Tips debug: Jika hasil tidak berubah, pastikan file sudah disimpan lalu refresh browser.

G. Rangkuman

  • if dipakai untuk kondisi tunggal.
  • if - elseif - else dipakai untuk banyak kemungkinan.
  • Operator logika membantu menggabungkan beberapa syarat.
  • Percabangan adalah dasar penting sebelum masuk form processing.
