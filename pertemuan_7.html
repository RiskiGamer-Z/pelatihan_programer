Pertemuan 7: Perulangan dan Array Dasar

Materi digabung agar siswa lebih cepat masuk project: looping dan array dipakai bersamaan untuk menampilkan data secara dinamis.

A. Tujuan Pertemuan

  • Menggunakan for, while, dan foreach.
  • Menyimpan banyak data dengan array numerik dan asosiatif.
  • Menampilkan data array dengan perulangan ke HTML.

B. Konsep Dasar Perulangan

Perulangan digunakan saat kita ingin menjalankan perintah yang sama berkali-kali tanpa menulis kode berulang secara manual.

Contoh kebutuhan nyata:

  • Mencetak nomor absen 1 sampai 36.
  • Menampilkan daftar siswa dari data array.
  • Mencetak tabel nilai dari banyak data.
Ingat: jika kondisi perulangan tidak berubah, program bisa masuk loop tak berujung (infinite loop).

Struktur Umum Perulangan

  • Nilai awal: titik mulai perhitungan.
  • Kondisi: syarat apakah loop lanjut atau berhenti.
  • Perubahan nilai: penambahan/pengurangan agar loop bergerak.

Jika salah satu dari tiga bagian ini tidak benar, program bisa berhenti terlalu cepat atau tidak berhenti sama sekali.

C. Jenis Perulangan di PHP

Ringkasan Pemakaian

  • for: saat jumlah putaran sudah diketahui.
  • while: saat bergantung kondisi yang berubah.
  • foreach: saat membaca data array.

1) For

Dipakai saat jumlah pengulangan sudah jelas (misal 1 sampai 10).

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Data ke-" . $i . "<br>";
}
?>

Penjelasan per kode:

  • for ($i = 1; ...): membuat variabel penghitung $i mulai dari 1.
  • $i <= 5: loop akan terus berjalan selama nilai $i belum melewati 5.
  • $i++: setiap putaran, nilai $i ditambah 1.
  • echo "Data ke-" . $i: menampilkan teks + nilai $i di tiap putaran.
  • <br>: pindah baris agar hasil tidak menumpuk dalam satu baris.

2) While

Dipakai saat perulangan bergantung kondisi tertentu.

<?php
$angka = 1;

while ($angka <= 5) {
    echo "Angka: " . $angka . "<br>";
    $angka++;
}
?>

Penjelasan per kode:

  • $angka = 1;: nilai awal.
  • while ($angka <= 5): cek kondisi sebelum masuk ke blok.
  • echo ...: menampilkan nilai saat ini.
  • $angka++;: wajib ada agar nilai berubah dan loop bisa selesai.

3) Foreach

Dipakai khusus untuk membaca isi array.

<?php
$namaSiswa = ["Andi", "Sinta", "Rudi"];

foreach ($namaSiswa as $nama) {
    echo "Nama siswa: " . $nama . "<br>";
}
?>

Penjelasan per kode:

  • $namaSiswa = [...]: array berisi 3 nama.
  • foreach ($namaSiswa as $nama): setiap item array disimpan sementara ke variabel $nama.
  • echo ... $nama: menampilkan item yang sedang dibaca.

4) Do-While (Contoh Tambahan)

Do-while mengeksekusi blok minimal 1 kali dulu, baru mengecek kondisi.

<?php
$nomor = 1;

do {
    echo "Nomor antrian: " . $nomor . "<br>";
    $nomor++;
} while ($nomor <= 3);
?>

Penjelasan per kode:

  • do { ... }: blok dijalankan terlebih dahulu.
  • $nomor++: menambah nilai agar mendekati kondisi berhenti.
  • while ($nomor <= 3): setelah blok jalan, barulah kondisi dicek.

D. Dasar Array untuk Menyimpan Data

Array adalah variabel yang bisa menyimpan banyak nilai sekaligus.

Jenis Array di PHP

  • Array numerik: indeks berupa angka otomatis (0, 1, 2, ...).
  • Array asosiatif: indeks berupa kata kunci (misal nama, kelas).
  • Array multidimensi: array di dalam array, cocok untuk data tabel.

1) Array Numerik

<?php
$jurusan = ["RPL", "TKJ", "DKV"];
echo $jurusan[0];
?>

Penjelasan per kode:

  • $jurusan = [...]: membuat array dengan 3 nilai.
  • $jurusan[0]: mengambil item pertama, yaitu RPL.

2) Array Asosiatif

<?php
$siswa = [
    "nama" => "Budi",
    "kelas" => "XI RPL",
    "nilai" => 88
];

echo $siswa["nama"];
?>

Penjelasan per kode:

  • "nama" => "Budi": menyimpan nilai Budi dengan kunci nama.
  • echo $siswa["nama"]: mengakses data berdasarkan kunci teks.

3) Array Multidimensi (Contoh Tambahan)

<?php
$kelas = [
    ["nama" => "Andi", "nilai" => 88],
    ["nama" => "Sinta", "nilai" => 92]
];

echo $kelas[1]["nama"];
?>

Penjelasan per kode:

  • $kelas[0] dan $kelas[1]: masing-masing menyimpan 1 data siswa.
  • $kelas[1]["nama"]: mengambil nama siswa pada index ke-1, yaitu Sinta.
  • Model ini umum dipakai untuk menampilkan data tabel dari banyak baris.

E. Contoh Integrasi Loop + Array + HTML Tabel

<?php
$dataSiswa = [
    ["nama" => "Andi", "kelas" => "XI RPL", "nilai" => 90],
    ["nama" => "Sinta", "kelas" => "XI TKJ", "nilai" => 85],
    ["nama" => "Rudi", "kelas" => "XI DKV", "nilai" => 78]
];
?>
<table border="1" cellpadding="8" cellspacing="0">
    <tr>
        <th>No</th>
        <th>Nama</th>
        <th>Kelas</th>
        <th>Nilai</th>
    </tr>

    <?php foreach ($dataSiswa as $index => $item): ?>
    <tr>
        <td><?php echo $index + 1; ?></td>
        <td><?php echo $item["nama"]; ?></td>
        <td><?php echo $item["kelas"]; ?></td>
        <td><?php echo $item["nilai"]; ?></td>
    </tr>
    <?php endforeach; ?>
</table>

Penjelasan per kode:

  • $dataSiswa: array multidimensi, tiap item mewakili satu siswa.
  • foreach ($dataSiswa as $index => $item): membaca data per siswa.
  • $index + 1: membuat nomor urut mulai dari 1.
  • $item["nama"], $item["kelas"], $item["nilai"]: membaca nilai berdasarkan kunci.
  • endforeach;: penutup gaya template PHP yang rapi saat bercampur HTML.
