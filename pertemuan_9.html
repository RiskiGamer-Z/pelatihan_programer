Fase 2: Backend Logic

Pertemuan 9: Mulai Project Pendataan Siswa Sederhana

Mulai dari pertemuan ini siswa membangun output 2 secara bertahap agar tidak menumpuk menjelang ujian.

A. Tujuan Pembelajaran

  • Memulai project pendataan siswa sederhana.
  • Menghubungkan form input dengan penyimpanan data sementara.
  • Menampilkan data siswa ke tabel menggunakan loop.
  • Membuat fitur reset data agar mudah saat uji coba.

B. Scope Project Hari Ini

  1. Form input data siswa: nama, kelas, jurusan.
  2. Simpan data ke $_SESSION (sementara, mudah untuk belajar).
  3. Tampilkan daftar data dalam tabel HTML.
  4. Tambahkan tombol reset untuk menghapus semua data latihan.

C. Alur Program Pendataan

  • Input form -> validasi -> simpan data.
  • Data lama tetap tampil saat halaman di-refresh.
  • Data ditampilkan dengan perulangan.
  • Reset data jika tombol "Hapus Semua Data" ditekan.

D. Konsep Inti yang Dipakai

1. Session untuk menyimpan data sementara

$_SESSION membantu kita menyimpan data selama browser belum ditutup. Cocok untuk latihan project awal sebelum masuk database.

<?php
session_start();

if (!isset($_SESSION['data_siswa'])) {
    $_SESSION['data_siswa'] = [];
}
?>

2. Form POST untuk input data

Gunakan method POST untuk mengirim data dari form ke proses PHP.

<form method="POST" action="">
    <input type="text" name="nama" placeholder="Nama" required>
    <input type="text" name="kelas" placeholder="Kelas" required>
    <input type="text" name="jurusan" placeholder="Jurusan" required>
    <button type="submit" name="simpan">Simpan</button>
</form>

3. Loop untuk menampilkan data ke tabel

Kita gunakan foreach agar setiap data siswa tampil otomatis sebagai baris tabel.

<?php foreach ($_SESSION['data_siswa'] as $index => $siswa): ?>
<tr>
    <td><?php echo $index + 1; ?></td>
    <td><?php echo $siswa['nama']; ?></td>
    <td><?php echo $siswa['kelas']; ?></td>
    <td><?php echo $siswa['jurusan']; ?></td>
</tr>
<?php endforeach; ?>

4. Validasi input wajib isi

Sebelum data disimpan, pastikan semua input tidak kosong.

<?php
$nama = trim($_POST['nama'] ?? '');
$kelas = trim($_POST['kelas'] ?? '');
$jurusan = trim($_POST['jurusan'] ?? '');

if ($nama === '' || $kelas === '' || $jurusan === '') {
    $pesanError = 'Semua field wajib diisi.';
}
?>

E. Contoh Implementasi Sederhana

Contoh Utama (Full File): Input + Tabel + Reset

Salin kode ini ke file pendataan_siswa.php, lalu jalankan dari browser. Kode sudah lengkap HTML + PHP sehingga tidak blank.

<?php
session_start();

if (!isset($_SESSION['data_siswa'])) {
    $_SESSION['data_siswa'] = [];
}

$pesan = '';

if (isset($_POST['simpan'])) {
    $nama = trim($_POST['nama'] ?? '');
    $kelas = trim($_POST['kelas'] ?? '');
    $jurusan = trim($_POST['jurusan'] ?? '');

    if ($nama === '' || $kelas === '' || $jurusan === '') {
        $pesan = 'Semua field wajib diisi.';
    } else {
        $_SESSION['data_siswa'][] = [
            'nama' => htmlspecialchars($nama),
            'kelas' => htmlspecialchars($kelas),
            'jurusan' => htmlspecialchars($jurusan)
        ];
        $pesan = 'Data berhasil disimpan.';
    }
}

if (isset($_POST['reset'])) {
    $_SESSION['data_siswa'] = [];
    $pesan = 'Semua data berhasil dihapus.';
}
?>
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pendataan Siswa Sederhana</title>
</head>
<body>
    <h2>Pendataan Siswa</h2>

    <?php if ($pesan !== ''): ?>
        <p><?php echo $pesan; ?></p>
    <?php endif; ?>

    <form method="POST" action="">
        <input type="text" name="nama" placeholder="Nama">
        <input type="text" name="kelas" placeholder="Kelas">
        <input type="text" name="jurusan" placeholder="Jurusan">
        <button type="submit" name="simpan">Simpan</button>
        <button type="submit" name="reset">Hapus Semua Data</button>
    </form>

    <hr>

    <?php if (count($_SESSION['data_siswa']) === 0): ?>
        <p>Belum ada data siswa.</p>
    <?php else: ?>
        <table border="1" cellpadding="8" cellspacing="0">
            <tr>
                <th>No</th>
                <th>Nama</th>
                <th>Kelas</th>
                <th>Jurusan</th>
            </tr>
            <?php foreach ($_SESSION['data_siswa'] as $index => $siswa): ?>
                <tr>
                    <td><?php echo $index + 1; ?></td>
                    <td><?php echo $siswa['nama']; ?></td>
                    <td><?php echo $siswa['kelas']; ?></td>
                    <td><?php echo $siswa['jurusan']; ?></td>
                </tr>
            <?php endforeach; ?>
        </table>
    <?php endif; ?>
</body>
</html>

Contoh Full File + CSS + Sedikit JavaScript

Contoh ini menambahkan tampilan card sederhana dengan CSS dan JavaScript kecil untuk konfirmasi sebelum reset data.

<?php
session_start();
if (!isset($_SESSION['data_siswa'])) {
    $_SESSION['data_siswa'] = [];
}

if (isset($_POST['simpan'])) {
    $nama = trim($_POST['nama'] ?? '');
    if ($nama !== '') {
        $_SESSION['data_siswa'][] = htmlspecialchars($nama);
    }
}

if (isset($_POST['reset'])) {
    $_SESSION['data_siswa'] = [];
}
?>
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pendataan Siswa + CSS + JS</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background: #f4f6fb;
            margin: 0;
            padding: 20px;
        }
        .card {
            max-width: 520px;
            margin: 0 auto;
            background: #ffffff;
            border-radius: 12px;
            padding: 18px;
            box-shadow: 0 8px 20px rgba(0, 0, 0, 0.08);
        }
        h2 { margin-top: 0; }

            display: flex;

            margin-bottom: 12px;
        }
        input[type="text"] {
            flex: 1;

            border-radius: 8px;
        }

            padding: 8px 12px;

            cursor: pointer;

        }

            font-size: 12px;
        }
    </style>
</head>
<body>
    <div class="card">
        <span class="badge" id="jumlahData">Total data: <?php echo count($_SESSION['data_siswa']); ?></span>
        <h2>Pendataan Siswa</h2>

        <form method="POST" action="">
            <input type="text" name="nama" placeholder="Nama siswa">
            <button type="submit" name="simpan">Tambah</button>
        </form>

            <button type="submit" class="btn-reset" name="reset">Hapus Semua Data</button>
        </form>

        <ul>
            <?php foreach ($_SESSION['data_siswa'] as $nama): ?>
                <li><?php echo $nama; ?></li>
            <?php endforeach; ?>
        </ul>
    </div>

    <script>
        function konfirmasiReset() {
            return confirm('Yakin ingin menghapus semua data?');
        }

        // Contoh JS kecil: ubah warna badge jika data lebih dari 5
        const total = <?php echo count($_SESSION['data_siswa']); ?>;
        const badge = document.getElementById('jumlahData');
        if (total > 5) {
            badge.style.background = '#fde68a';
            badge.style.color = '#92400e';
        }
    </script>
</body>
</html>
